    Project Methodology

    This personal project implements a hybrid Neuro-Symbolic forecasting pipeline. While standard machine learning architectures (GRU, XGBoost, RF) capture the complex atmospheric interactions, they are prone to physically inconsistent errors at longer horizons. By routing predictions through a symbolic execution filter built on Z3, we guarantee that all predictions comply with basic physical laws of atmospheric dry-out, aerosol light attenuation, and stagnant radiation fog accumulation.
